Asignación

Desarrollo de una página web con el framework Django

Desarrollo de una página web con el framework Django

Competencias:

en el conocer: Nuevas tecnologías y conocimiento en el desarrollo full-stack de una aplicación web.

en el hacer: Procedimiento de vanguardia utilizados por múltiples empresas de primer mundo.

en el ser: Mediante el uso de GitHub se podrá controlar las versiones de las aplicaciones y verificar el desempeño y mejora de los proyectos.

 

Indicaciones del proyecto.

 

                                 Detallar el desarrollo del proyecto

 

Realizará un proyecto en el cual se desarrollará una página web especializada en ventas de productos tecnológicos y review de los productos con el framework Django, se utilizará MySQL para el manejo de base datos y como lenguaje de programación principal se utilizará Python. Además, en la sección de front-end se usará HTML, CSS para la edición visual de la página web.

Objetivos de la asignación:

  • Desarrollar una aplicación web con el framework Django.
  • Implementar base de datos que maneje la información de los productos tecnológicos.
  • Diseñar la sección visual de la aplicación web con HTML y CSS.
  • Actualizar y subir el código desarrollado del proyecto en GitHub por medio de la aplicación GitKraken.
     

Tecnologías a utilizar:

  • Python
  • Django
  • MySQL
  • HTML
  • CSS
  • GitHub
  • GitKraken
     

Entrega de la asignación

 

La asignación se debe subir a la plataforma GitHub y se enviará a Moodle el link de acceso a GitHub.

 

Fecha de entrega: 20 de octubre

Lenguajes Front End – Cliente

En palabras simples, todo lo que el usuario de una aplicación está viendo es una interfaz (frontend), como menús de navegación, plantillas y botones de la aplicación, etc. JavaScript, CSS, React y HTML son los lenguajes de frontend ampliamente reconocidos.

Javascript

JavaScript es un lenguaje de programación interpretado, dialecto del estándar ECMAScript. Se define como orientado a objetos, ​ basado en prototipos, imperativo, débilmente tipado y dinámico.

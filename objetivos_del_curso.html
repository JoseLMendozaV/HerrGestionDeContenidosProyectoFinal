Objetivos del curso

  • Objetivo General
    • Desarrollar aplicaciones utilizando una plataforma y entorno de programación web o empleando una herramienta de gestión de contenido
  • Objetivos Específicos
    • Conocer la arquitectura de las aplicaciones web.
    • Identificar la estructura para la creación de una aplicación web.
    • Identificar las diferentes herramientas requeridas para diseñar, construir y mostrar una aplicación web.
    • Desarrollar el diseño y creación de aplicaciones web a través de HTML5 y CSS e integrarlos.
    • Desarrollar aplicaciones web que manejen base de datos a través de un lenguaje de programación.
    • Aplicar una herramienta gestora de contenido que permita crear, diseñar y mostrar una aplicación web.

Queridos estudiantes

Esta asignatura tiene como meta que los estudiantes obtengan los conocimientos habilidades y destrezas necesarias para el desarrollo e integración de aplicaciones basadas en un entorno web, integrando lenguajes de programación, base de datos y manejo de servidores. Para lograr esto, el estudiante debe manejar un lenguaje de programación orientado a la web.
